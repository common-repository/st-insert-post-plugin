== Description ==

A simple front end post form for all users, and as a bonus the ability to list subpages.

Just insert [st_insert_post] into a page to display the front end post form.
To list a pages subpages, just insert [stls] into a page.
Simple eh?


== Installation ==

1. Unzip plugin files and upload them under your '/wp-content/plugins/'

2. Activate plugin


== Upgrade Notice ==

Upgrade using the automatic upgrade in Wordpress Admin
